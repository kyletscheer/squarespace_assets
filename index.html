Technical Requirements & Hardware Warnings
  • Desktop Required: Designed for high-volume file handling. Will likely fail on mobile.
  • Available Storage: Ensure sufficient disk space for your full library before starting.
  • RAM Management: 4–8 GB of free system RAM recommended for large libraries. Higher concurrency settings need more RAM.
  • Chrome Settings (fallback mode): If no folder is selected, go to Settings → Downloads and turn OFF "Ask where to save each file before downloading."
Images Only — Video Download Guide

⚠️ This utility processes IMAGES ONLY. Due to Squarespace's backend loading structure, video files cannot be downloaded through this tool.

If your Squarespace Asset Library contains videos, you can download them separately using the Video DownloadHelper browser extension:

  • Install the extension: Video DownloadHelper on Chrome Web Store or Video DownloadHelper on Firefox Add-ons
  • Visit your Squarespace Asset Library in your web browser
  • Click on each video to go to the video page
  • Click the Video DownloadHelper icon in your browser toolbar when hovering over a video
  • Select your preferred quality and download location

Video Tutorial: I've prepared a tutorial video showing exactly how to use Video DownloadHelper. Check back soon or watch our main tutorial (here) for updates.

Smart Format Detection & .webp Files

This utility specifically requests your original uploaded formats (PNG/JPG) from Squarespace. However, modern CDNs often "force" a conversion to .webp to save space and speed up loading.

  • Automatic Correction: We perform a real-time "MIME-check" on every download. If the server ignores our request and sends a WebP, the tool automatically renames the file to .webp.
  • Why this matters: If we saved a WebP as a .png, the file would be "corrupt" and rejected when you try to re-upload it or edit it in apps like Preview.
  • High Compatibility: These .webp files are high-quality and natively supported by Squarespace, Shopify, WordPress, and all modern Operating Systems.
  • Pro Tip: If you strictly need a .png for a legacy project, use a bulk converter on your Mac (like sips or Quick Actions) after the download completes.
